\subsection{Detailed Description}
An Abstract class used as a base to create different monitors. 

Has three methods that must be overriden\-: \begin{DoxyVerb}  1) poll(), the function that returns the current value of whatever resource you're looking at.
  2) minimum(), which must return the lowest value that the resource could take on.
  3) maximum(), which must return the largest value that the resource could take on. \end{DoxyVerb}
